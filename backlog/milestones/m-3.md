## Description

Build the IPC daemon that wraps libwlblur and provides blur-as-a-service to Wayland compositors. Implements Unix socket protocol with DMA-BUF file descriptor passing.

**Deliverables:**
- IPC protocol definition
- Unix socket server
- Blur node registry
- Client state management
- systemd unit file
